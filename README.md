### Prerequisites

- Node version 20.10.0

### Cloning the repository

```shell
git clone https://github.com/betomossmann/next-auth.git
```

### Install packages

```shell
pnpm i
```

### Setup .env file

```js
DATABASE_URL=
DIRECT_URL=

AUTH_SECRET=

GITHUB_CLIENT_ID=
GITHUB_CLIENT_SECRET=

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=

RESEND_API_KEY=

NEXT_PUBLIC_APP_URL=
```

### Setup Prisma

```shell
pnpm prisma generate
pnpm prisma db push
```

### Start the app

```shell
pnpm dev
```
